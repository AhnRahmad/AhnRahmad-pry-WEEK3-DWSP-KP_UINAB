# 📚 Perpustakaan Digital UIN Antasari Banjarbaru

![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

&gt; Sistem Katalog Buku Digital berbasis web dengan fitur pencarian real-time dan manajemen koleksi untuk Perpustakaan UIN Antasari Banjarbaru.

## ✨ Fitur Utama

- 🔍 **Pencarian Real-Time** - Filter buku berdasarkan judul, penulis, atau kategori secara instan
- ➕ **Tambah Koleksi** - Form modal untuk menambahkan buku baru dengan validasi data
- 🏷️ **Filter Kategori** - Sortir buku berdasarkan kategori yang tersedia
- 📊 **Statistik Dashboard** - Counter koleksi buku yang update otomatis
- 📱 **Responsive Design** - Tampilan optimal di desktop, tablet, dan mobile
- 🎨 **UI Modern** - Desain elegan dengan tema warna biru-kuning emas profesional

## 🚀 Teknologi

- **Frontend:** HTML5, CSS3 (Custom Properties, Flexbox, Grid)
- **Styling:** Google Fonts, Font Awesome Icons
- **Logic:** Vanilla JavaScript (ES6+)
- **Storage:** In-Memory (dapat dikembangkan ke LocalStorage/Database)

## 📋 Struktur Data Buku

| Field | Keterangan |
|-------|-----------|
| Judul | Nama lengkap buku |
| Penulis | Nama pengarang |
| Tahun | Tahun terbit |
| Kategori | Genre/klasifikasi buku |
| Cover | Gambar sampul buku |

## 🎯 Cara Penggunaan

1. Buka file `index.html` di browser
2. Gunakan kolom pencarian untuk mencari buku
3. Klik "Tambah Koleksi" untuk menambah buku baru
4. Klik gambar buku untuk melihat cover full-size
